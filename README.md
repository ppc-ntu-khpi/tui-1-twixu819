# UI Lab 1
![](terminal-icon.png)
![](gui-icon.png)

В цій практичній роботі я розібрався з консольним графічним інтерфейсом ```TUI``` за допомогою бібліотеки Jexer.
Було виконано завдання на 4, за допомогою оператора instanceof реалізовано динамічну перевірку та відображення типу рахунку клієнта ```SavingsAccount``` чи ```CheckingAccount``` в інтерфейсі програми.

Для запуску програми треба ввести команду в терміналі: java -cp "Lab 1 -TUI;jars/*" com.mybank.tui.TUIdemo
